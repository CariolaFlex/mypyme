# Decisiones Tributarias para la Operación del SaaS

**Versión:** 1.0 · **Fecha:** 2026-06-13

> Aplica a la operación de Vectium como emisor del SaaS, no al cliente final.

---

- **Régimen recomendado:** Pro Pyme Transparente (si persona natural o socios personas
  naturales) — sin IDPC, se declara directamente en Global Complementario. Más simple
  mientras el negocio es pequeño.

- **IVA del SaaS:** el servicio de software es afecto a IVA. Hay que emitir
  boleta/factura electrónica por cada cobro de suscripción. Flow paga el neto; tú emites
  el documento tributario.

- **F29 mensual:**
  - Débito = IVA de suscripciones cobradas.
  - Crédito = IVA de costos (Vercel, Supabase si emiten factura chilena, hosting, etc.).

- **Patente municipal:** requerida para operar como empresa de software/servicios.

- **Obligaciones SII:** inscribirse como emisor electrónico, emitir boleta dentro de
  1 hora de cada transacción.
